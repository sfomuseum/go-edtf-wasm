EDTF Parser

EDTF level 0, 1 and 2 EDTF strings are accepted, with the exception of specific features noted in the go-edtf documentation.

This webpage demonstrates the go-edtf parser functionality exported to JavaScript as a WebAssembly binary. The source code for this webpage is available from SFO Museum's GitHub account:
