EDTF Parser

Enter a valid Extended DateTime Format (EDTF) string. Level 0, 1 and 2 EDTF strings are accepted, with the exception of specific features noted in the go-edtf documentation.

This webpage demonstrates the go-edtf parser functionality exported to JavaScript as a WebAssembly binary. Please keep in mind that the go-edtf package is still currently in development and, by extension, so is this webpage.

The source code for this webpage is available from SFO Museum's GitHub account:
